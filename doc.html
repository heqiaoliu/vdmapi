1. Request Format

PMAPI Request

Http request vdm.sdsu.edu/data/api/pmapi.php?type=PM_REQUEST_TYPE& param=YOUR_PARAMATERS[&mid=YOUR_MESSAGE_ID(optional)(customized)]

PMAPI Return Value

The return value of pmapi is in JSON Format.

attributetypevalue
dataarray/object(associate array/ dictionary)reuturn data based on request
successbooleantrue if the request met no error/false if error occours
error_messageString
error_codeInteger
midIntegercustomized message id
